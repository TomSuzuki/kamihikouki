## 紙ヒコーキ 
単純な障害物避けゲームです。  
最高記録の保存は行っていません。  

## 操作方法
操作はすべてマウス  
STARTをクリックで開始、ゲームオーバー後はクリックでタイトルへ戻れます。  

## ゲームの説明
黒いボールに当たるか壁に当たるか、又は、角度をつけすぎて墜落するとゲームオーバーです。  
ボールにはプレイヤーを追いかけるものとランダムなものの2種類が存在します。
